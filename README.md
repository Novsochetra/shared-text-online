# ✨ TextShare - Instant Text Sharing via URL

**Share text between devices instantly - no sign-up, no server, just a link!**

## 🎯 What is TextShare?

TextShare is a simple, privacy-focused web app that lets you share text between your devices (computer, phone, tablet) using just a URL. Type your text, generate a link, and open it anywhere - your text appears instantly!

## 🚀 Key Features

- **📱 Cross-Device Sharing** - Write on desktop, read on mobile (or vice versa)
- **🔗 URL-Based** - Everything is encoded in the link itself
- **🔒 Privacy First** - No data stored on servers, all in the URL
- **⚡ Instant Access** - No loading, no accounts, just paste and view
- **📋 One-Click Copy** - Copy generated links instantly to clipboard
- **🌐 Works Offline** - After first load, works without internet
- **🎨 Clean UI** - Simple, modern design that works on any screen size
- **💾 No Storage Limits** - Share as much text as your browser can handle
- **🔓 Open Source** - Fully transparent, no hidden tracking

## 🛠️ How It Works

1. **Type** - Enter your text in the textarea
2. **Generate** - Click "Generate Link" to create a shareable URL
3. **Share** - Copy the link and open it on any device
4. **View** - Your text appears instantly, no downloads needed!

**Technical Magic:** Your text is encoded using Base64 and embedded directly in the URL. No server uploads, no databases - just pure client-side JavaScript!

When you open a TextShare link, the text automatically appears - no extra clicks needed.

### Edit & Reshare
Received a link? You can edit the text and generate a new link instantly.

## ⚠️ Limitations

- Very long text (>10,000 chars) may create extremely long URLs
- Some apps may truncate very long URLs
- Text is not encrypted (visible in URL)
- Not suitable for highly sensitive information

## 🎓 Technical Details

- **Size:** Single HTML file (~5KB)
- **Dependencies:** None! Pure vanilla JavaScript
- **Encoding:** Base64 with UTF-8 support
- **Framework:** None - just HTML, CSS, JavaScript
- **Storage:** None - everything in URL parameters

## 📞 Support

Having issues? Common solutions:
- **Link too long?** - Reduce text length or use URL shortener
- **Text not appearing?** - Check if URL is complete/untruncated
- **Copy not working?** - Try manual selection and copy
- **Emoji issues?** - Should work! If not, report the specific emoji
